# Atheer's Unit Converter

This is a simple CLI that allows users to convert measurements into different units.

![screenshot of CLI](https://res.cloudinary.com/dl4pcxjxz/image/upload/v1605444582/Screenshot_2020-11-15_at_12.49.29_hbrufj.png "A screenshot of atheers-unit-converter")

## Install/Run Instructions

to install this package globally, enter the following into terminal 

**npm install -g atheers-unit-converter** 

to run the package enter the following into terminal (from anywhere)

**convert**

NPM package can be found online [here](https://www.npmjs.com/package/atheers-unit-converter "atheers-unit-converter - npmjs")

### Supported Conversions

| Unit      | Supported Conversion |
| ------------- |:-------------:|
| cm | m |
| m | cm |
